# Razor-Consumir-SOAP

Exemplo de consumo de WebServices utilizando SOAP em localhost.

## Requisitos

Criação do servidor SOAP (http://localhost:54222/PessoaService.asmx)   

A criação de exemplo encontra-se nesse link (https://github.com/Marcelofazan/soap-provedor-database) 

# O que você vai encontrar neste projeto

- **CRUD** - Conjunto de quatro operações essenciais que permitem a manipulação de dados persistentes (criar, ler, atualizar e excluir).
- **Client SOAP** - Manipulação de métodos disponíveis para enviar/receber dados.

## Execução da aplicação

Para executar a aplicação é necessário, seguir somente o requesito.
