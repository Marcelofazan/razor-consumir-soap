### Razor-Consumir-SOAP

Exemplo de consumo de WebServices utilizando SOAP em localhost.

### Requisitos

Criação do servidor SOAP (http://localhost:54222/PessoaService.asmx)   
- Disponivel em [soap-provedor-database](https://github.com/Marcelofazan/soap-provedor-database/archive/refs/heads/main.zip)

#### O que você vai encontrar neste projeto

- **Client SOAP** - Manipulação de métodos CRUD para enviar/receber dados.

## Execução da aplicação

Para executar a aplicação é necessário, seguir somente o requesito.
